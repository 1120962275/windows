# windows
ProxySU for windows v2ray install tools

学习C#用来练手的小工具。

编译环境Visual Studio 2017 

使用WPF界面

支持的VPS系统为：

CentOS 7/8 

Debian 8/9/10

Ubuntu 16.04及以上

目前已支持生成用于

1>v2ray官方程序配置文件(客户端配置)

2>v2rayN客户端导入二维码和网址

3>Shadowrocket(ios)导入二维码和网址
